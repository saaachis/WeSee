About WeSee


We See - A Helper for the Visually Impaired!


Simply, it is built as an a web application for individuals with low vision
capabilities to excel in daily tasks and requirements.

The project is aimed to address the challenges faced by visually impaired individuals
by leveraging cutting-edge technologies, primarily based in Computer Vision and
Digital Image Processing, specifically built upon the foundations of Deep Learning.

Through the integration of state-of-the-art technologies in Artificial Intelligence
and Machine Learning, this project contributes to the inclusive integration of
individuals with visual disability into our society by fostering independent access to information and promoting equal opportunities.


The System Design and Development.


This web application involves the functionality of many technologies for its various uses.

Thus, WeSee is created with the following key structures:

- camera input and pre-processing

- object detection and recognition

- scene understanding and description

- audio output and feedback

- user interface and interaction
